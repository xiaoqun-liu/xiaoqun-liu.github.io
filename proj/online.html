Threshold Moving for Online Class Imbalance Learning with Dynamic Evolutionary Cost Vector

Peijia Qin, Shuxian Li, Xiaoqun Liu, Zubin Zheng, Sun Yew Chong(SUSTech)

Research Background
Online learning faces challenges related to limited time and memory resources in real-time data streams. Online class-imbalance learning refers to scenarios in which samples from minority classes are significantly fewer than those from majority classes, adversely affecting predictive performance, particularly for minority class predictions. Existing methods for online class-imbalance learning rely on assumptions regarding the expected enhancement of minority classes, which are often temporary and hard-coded into the model parameters.
Paper on open reivew.

Definition of Cost Vector

A cost vector is a multidimensional array in which each dimension represents a class. The values of the dimensions indicate the cost of misclassifying one class as another. In cases of class imbalance, the misclassification cost for minority class samples is typically set higher to enhance the model's ability to recognize these minority classes.

Uses of Cost Vector

  • Adjusting Decision Boundaries: By applying the cost vector to the predicted outputs of a classifier, the decision boundaries can be adjusted, making the model more sensitive to minority class samples.
  • Optimizing Performance Metrics: The cost vector can assist the model in optimizing specific performance metrics (such as accuracy, recall, F1 score, etc.), especially in scenarios with uneven class distribution.
  • Adapting to Concept Drift: In online learning environments, data distributions may change over time (concept drift). The cost vector can be dynamically updated through evolutionary algorithms to adapt to these changes.
  • Implementing Two-Layer Optimization: In the Online Evolving Cost Vector (OECV) framework, the cost vector serves as a hyperparameter for the upper-layer optimization problem, working together with the lower-layer online classifier to form a two-layer optimization structure. The lower-level classifier provides initial predictions, while the cost vector refines these predictions.
  • Enhancing Prediction Accuracy for Minority Classes: In imbalanced data streams, minority class samples are often overlooked by the model. By appropriately setting the cost vector, the model's focus on minority class samples can be increased, thereby improving prediction accuracy for these samples.
