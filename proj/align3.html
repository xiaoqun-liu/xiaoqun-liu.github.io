LLM Security

An illustration of how CTRL works, where ➀➁➂ represent generated texts through output sampling. In this case, ➀ has lower perplexity, while ➁ demonstrates poor helpfulness. As a result, the beam search selects ➂ for the next round of output sampling. Perplexity is measured by a LLM that needs to be robustified, and helpfulness is rated by GPT-4o using prompts.
